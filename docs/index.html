Overview

Rippley is a modern Neo-Glyph agent system built with FastAPI and advanced agent orchestration capabilities. It provides a clean, modular architecture for creating and managing intelligent agents.

Architecture

The system is organized into the following core modules:

  • rippley_viewer/ - FastAPI web interface and templating
  • agent_core/ - Core agent functionality and management
  • config/ - Configuration files for glyph specifications and NL rules
  • docs/ - Documentation and guides

Quick Start

Prerequisites: Python 3.8+, FastAPI, Uvicorn

Installation

pip install -r requirements.txt

Running the Application

# Option 1: Using the entry point python make_rippley.py # Option 2: Direct uvicorn command uvicorn rippley_viewer.app:app --host 0.0.0.0 --port 8000 --reload

Agent Core Modules

AgentFactory

Creates and manages different types of Neo-Glyph agents.

from agent_core import AgentFactory factory = AgentFactory() agent = factory.create_agent("neo_glyph", {"id": "my_agent"})

TaskRunner

Manages and executes tasks for agents with concurrent processing capabilities.

from agent_core import TaskRunner runner = TaskRunner(max_concurrent_tasks=5) task = runner.add_task("task_1", "process_data", {"data": "example"})

MemoryLink

Provides agent memory and context storage with categorization and search capabilities.

from agent_core import MemoryLink memory = MemoryLink("agent_1") memory.store("context", "conversation_data", category="conversation")

Configuration

Glyph Specification (config/glyph_spec.json)

Defines agent capabilities, patterns, and integration points for Neo-Glyph behavior.

Natural Language Rules (config/nl_rules.json)

Configures natural language processing, intent recognition, and response generation.
